# TP Cryptographie - Algorithme Ariel-Hash

Ceci est un algorithme de hachage personnalisé pour mon TP.

## Fonctionnement
- **Taille :** 64 bits (16 caractères hexadécimaux).
- **Méthode :** Utilisation d'un XOR, d'une multiplication par un nombre premier et d'un décalage circulaire (bit rotation).

## Comment tester
Lancer le script avec `python hachage_ariel.py` et saisir un texte.
